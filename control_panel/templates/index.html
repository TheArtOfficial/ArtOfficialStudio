Download CivitAI Models

How to use the CivitAI Downloader
  1. Go to CivitAI and find the model you want to download
  2. Right Click the "Download" button on the model page and copy the URL
  3. If the link has a "?" in it, remove the "?" and everything after it
  4. Paste the URL in the field below
  5. Select the appropriate model type (LoRA, Checkpoint, etc.)
  6. Enter a filename (without extension) to save the model as
  7. Enter your CivitAI token
  8. Click "Download" to start the download process

Download from HuggingFace

How to use the HuggingFace Downloader
  1. Go to HuggingFace and find the model you want to download
  2. Navigate to the specific file you want (e.g., a .safetensors file)
  3. Copy the direct URL to that file (right-click and "Copy Link")
  4. Paste the URL in the field below
  5. If there is a "?" in the URL, remove the "?" and everything after it
  6. For private or gated models, enter your HuggingFace token
  7. Specify a download path if you want to save to a specific location
  8. If no path is specified, it will download to the default models directory
